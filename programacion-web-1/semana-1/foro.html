Explicación:

Para el título se ha usado la etiqueda h1 ya que es la mejor opción para representar el título de la página.

Para el subtítulo se ha usado la etiqueda h3 ya que esta etiqueta tiene menor peso que el h1 en cuanto a título o encabezados, por lo que es ideal para subtítulos. También podría usarse un h2 para subtítulos.

Para los párrafos, lo más común es usar la etiqueta p.

Para los estilos se usa el atributo style, que nos permite insertar cualquier sentencia css dentro de ella.
